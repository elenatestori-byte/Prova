# Search Everything

Unlike Windows' [built-in search, Search Everything](https://www.voidtools.com/Everything-1.4.1.895.x64.zip) instantly indexes file and folder names on NTFS disks, giving you real-time results. In seconds, you can locate suspicious files, hidden folders, and programs wherever they are located on the system.

### Filters and Operators <a href="#filtri-e-operatori" id="filtri-e-operatori"></a>

The power of [Search Everything](https://www.voidtools.com/Everything-1.4.1.895.x64.zip) lies in its advanced search syntax, which allows you to combine filters and operators to get highly accurate results.

**Common Filters**

* `ext:` Find files of a specific extension, as if to search for all Java files.`ext:jar`
* `size:` Filter by file size. For example, to find files of 1 megabyte or larger, or for smaller files.`size:>=1mbsize:<10kb`
* `date:` Search for files based on their date modified, created, or accessed. For example, find files that were changed today. You can also use a date range, such as .`date:todaydate:2025/09/01-2025/09/03`

**Boolean Operators**

Boolean operators allow you to combine multiple terms for a more specific search.

* `AND` (represented by a space): Find files that contain all the terms. Useful for targeted searches such as .`cheat.exe AND Downloads`
* `OR` (represented by): Finds files that contain at least one of the terms. An example is .`|cheat.jar | autoclicker.exe`
* `NOT` (represented by): Excludes a term. For example, it will find all files that don't have "minecraft" in the name or path.`!*.jar !minecraft.jar`

By combining these filters and operators, you can quickly refine your search and focus on what's really relevant to your survey.
