# Using System Informer

## Using System Informer

If Prefetch files and other Windows registries give you a historical snapshot of activity, [System Informer](https://systeminformer.sourceforge.io/downloads.php) gives you a detailed, real-time view of everything that's happening on your computer. It is an advanced replacement for the Windows Task Manager, providing granular control over processes, open files, and network connections. Its user-friendly interface and powerful features make it an essential tool for finding cheats.

### **What Is Kernel Mode and Why Is It Crucial?** <a href="#cose-il-kernel-mode-e-perche-e-fondamentale" id="cose-il-kernel-mode-e-perche-e-fondamentale"></a>

The kernel is the core of the operating system. It operates at a higher level of privilege (kernel mode) than regular applications (user mode). While user-mode programs have limited access to system resources for security reasons, kernel-mode programs have full and unrestricted access.

* **How We Use It**: By enabling kernel mode in System Informer, the program installs a driver that allows it to operate at a higher level of privilege. This way, they can see processes and tasks that would otherwise remain hidden.

### **How to Enable Kernel Mode** <a href="#come-attivare-il-kernel-mode" id="come-attivare-il-kernel-mode"></a>

To enable this mode, you must launch System Informer with administrator privileges. From the menu, go to Options > Enable Kernel Mode Driver. The program may require a restart to apply the changes.
