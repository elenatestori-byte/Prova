# Sources and References

### 1. Forensic Software and Utilities <a href="#id-1.-software-e-utilita-forensi" id="id-1.-software-e-utilita-forensi"></a>

**WinPrefetchView**, **LastActivityView**, **USBDeview**, **Search Everything**, **AlternateStreamView**, **HashMyFiles**: The definitions and functionalities of these tools have been verified through the official documentation on the developers' websites (NirSoft and Voidtools).

**PECmd**, **Registry Explorer**, **bstrings**: Information on these tools comes from reference guides and official documentation provided by **Eric Zimmerman.**

**System Informer**, **JournalTrace**: The concepts related to these tools are based on their official documentation, **on the GitHub** pages of the projects and on dedicated discussion forums.

**Red Lotus** scripts, **Spokwn Path Parser** and **JournalTrace** and **HabibiModAnalyzer**: The information and commands were provided directly from the projects' documentation on GitHub or Discord, which describes how they work and how they are used.

**Luyten**: The concepts related to these tools have been verified through the official documentation on their respective websites and GitHub pages.

### 2. Technical and Operating System Documentation <a href="#id-2.-documentazione-tecnica-e-del-sistema-operativo" id="id-2.-documentazione-tecnica-e-del-sistema-operativo"></a>

**Prefetch**, **Registry**, **Event Viewer**, **USN Journal**, **Task Scheduler**, **WMIC:** Understanding of these system artifacts has been verified through **official Microsoft** documentation, **Windows** technical manuals, and academic and professional publications on computer forensics.

**Processes** (**CSRSS**, **MsMpeng**, **PlugPlay**, **PcaSvc**), **Fileless Injection**: The definitions of these processes and techniques have been consulted on blogs and websites (e.g. **SANS Institute**, **CISA)** and technical reference documentation.

**File System** (**FAT32**, **exFAT**): File system features, such as the absence of the **USN Journal**, have been confirmed through **official Microsoft** documentation and educational resources on operating system architecture.

**Ghost Client**, **Injection Client**, **External Client**: The concepts and detection techniques related to these categories of cheats are based on knowledge shared within Minecraft forums and communities.
