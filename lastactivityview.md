# LastActivityView

[LastActivityView](https://www.nirsoft.net/utils/lastactivityview.zip) is a tool from NirSoft that creates a detailed history of the user's recent activities. Unlike real-time monitoring tools, LAV collects information from various Windows sources to give you a historical view, allowing you to reconstruct a timeline of events even after they have happened.

### **What it shows and how it works** <a href="#cosa-mostra-e-come-funziona" id="cosa-mostra-e-come-funziona"></a>

Just launch [LastActivityView](https://www.nirsoft.net/utils/lastactivityview.zip) to have it scan your system. The program aggregates data from several sources, including:

* File Prefetch
* Registry Keys:
  * Recent file history
  * File Opening and Saving History (MRU)
  * UserAssist
* Windows Event Log
* Recycle Bin Information
* Application Crash Reports

These sources provide [ LAV](https://www.nirsoft.net/utils/lastactivityview.zip) with the information it needs to create a log of actions such as file execution, folder opening, software installation, and system startup or shutdown events.`.exe`
