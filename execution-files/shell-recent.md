# Shell:Recent

Analyzing recently opened files is one of the quickest methods in a screenshare. Windows automatically creates shortcuts to files and folders that a user has opened, making it possible to quickly inspect their activity.

### How it works and where to find it <a href="#come-funziona-e-dove-trovarlo" id="come-funziona-e-dove-trovarlo"></a>

The recent files folder, accessible via the , command contains shortcut files () that point to the original files. Each link stores crucial metadata, such as the file's path and its timestamps.`shell:recent.lnk`

To quickly access this folder:

1. Press to open the Run window.`Win + R`
2. Type and press Enter.`shell:recent`

This will open a folder showing links to all recently opened files.

It is important to note that a user can easily empty the folder. If the folder is empty or if the files inside it are dated, it is not proof of innocence, but rather a sign that the player may have tried to hide their tracks.`shell:recent`
