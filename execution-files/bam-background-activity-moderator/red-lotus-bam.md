# Red Lotus Bam 🪷

Manual analysis of BAM from the system registry can be complex and time-consuming. For this, many screensharers rely on automated scripts that extract information in seconds.

The  script is a tool in PowerShell that automates the extraction of data from BAM and displays it in a graphical user interface (GUI).

### Script: <a href="#script" id="script"></a>

RedLotus BAM<a class="button secondary">Copy</a>

```
powershell Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass && powershell Invoke-Expression (Invoke-RestMethod https://raw.githubusercontent.com/PureIntent/ScreenShare/main/RedLotusBam.ps1)
```
