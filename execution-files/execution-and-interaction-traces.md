# Execution and Interaction Traces

Within a screenshare, the most conclusive evidence that a player has used a file, such as a client cheat, lies in its execution. Starting a file not only leaves a trace in the computer's memory, but also generates digital artifacts in the operating system, which can be analyzed to confirm suspicious activity.

The Windows operating system offers several methods for tracking and verifying the execution of a file. The analysis of these "registers" is a fundamental step for an accurate investigation.

In this section, we'll explore the main tools and techniques that are most effective for demonstrating the startup of a file, such as parsing the Prefetch, the shell:recent directory, and other methods that we'll go into more detail.
