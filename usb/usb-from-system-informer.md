# USB from system informer

A [little-known but extremely useful feature of the System Informer](https://systeminformer.sourceforge.io/downloads.php) is its ability to show disk devices that have been removed. This allows you to check if a USB stick or external drive was disconnected just before your screenshare.

### How to Check Removed USBs <a href="#come-controllare-le-usb-rimosse" id="come-controllare-le-usb-rimosse"></a>

1. Open [System Informer](https://systeminformer.sourceforge.io/downloads.php).
2. Go to the "Options" menu and select "Disk Devices".
3. In the window that opens, you will see the list of all the disks. Click on the disk you want to inspect
4. Select "Details" and in the dropdown named "Properties".
5. Scroll down until you find the "Last Removal Date" entry.

This will show you the last time a device was disconnected. A removal time that coincides with the start of your screenshare is a strong clue that the user has unplugged a thumb drive to hide evidence.
