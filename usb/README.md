# USB

USBs, like thumb drives, are a common tool for cheats. Some players choose not to "install" cheats on their computer, preferring to run them directly from a flash drive to avoid execution and installation tests. Therefore, USB analysis is an essential step in any screenshare. A key aspect to consider is that flash drives formatted in **FAT32** and **exFAT** do not have the **USN Journal**, a log that tracks changes to files.
